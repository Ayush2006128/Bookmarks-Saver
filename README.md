A lightweight and efficient Progressive Web App (PWA) designed to help you save and manage your favorite web bookmarks locally in your browser.

---

## 🌟 Features

- **Save Bookmarks:** Store URLs with custom names for quick access.
- **Easy Management:** View and remove bookmarks with a single click.
- **Confirmation Dialogs:** Safety check before removing a bookmark.
- **Progressive Web App (PWA):** Installable on your device for a native-like experience.
- **Offline Support:** Access the app even without an internet connection, thanks to a robust service worker.
- **Clean UI:** Simple and responsive design for all screen sizes.

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser to run this application.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/bookmarks-saver.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd bookmarks-saver
   ```
3. **Open `index.html` in your browser:**
   - Double-click the file in your file explorer.
   - Or use a local development server:
     ```bash
     npx serve .
     ```

## 🛠️ Built With

- **HTML5:** Semantic structure for the web.
- **CSS3:** Modern styling with a clean aesthetic.
- **JavaScript (ES6+):** Dynamic functionality and DOM manipulation.
- **Service Workers:** For caching and offline functionality.
- **Local Storage API:** To persist bookmarks locally.

## 📱 PWA Support

This app is fully compatible with PWA standards. You can install it on:
- **Desktop:** Chrome, Edge, and Safari.
- **Mobile:** Android (via Chrome) and iOS (via "Add to Home Screen" in Safari).
